# SmallInteractive3D

Move a box to the pressure plate and unlock the lever. Interact with lever to open the door

## Game Controllers
Forward: ```W or ^```<br />
Left: ```A or <-```<br />
Backward: ```S or v```<br />
Right: ```D or ->```<br />
Interact: ```E```<br />
